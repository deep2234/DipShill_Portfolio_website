Projects

KindDoner – Cloud-Powered Donation Platform

A full-stack serverless web app built solo, KindDoner enables users to submit donation details (both physical and monetary) through a clean, responsive UI. The frontend is crafted with HTML, CSS, and JavaScript, while the backend leverages AWS Lambda, API Gateway, and DynamoDB to securely handle submissions. Physical donations capture dynamic item details and contact info, while monetary contributions use a simple, mobile-friendly UPI QR system. This project strengthened my skills in serverless architecture, REST API development, and AWS services. 🔗[GitHub Repo- Click Here]
